Steps to setup:
  ###### cd into project directory
  ###### run bundle install
  ###### rake db:migrate
  ###### change mysql password in database.yml according to your system
  ###### run rails s
  ###### visit http://localhost:3000/
  ###### you will need to signup to start creating tasks.
  
Steps to run specs:
  # to run model specs
  rspec spec/models
  
  # to run feature specs
  rspec spec/features/
